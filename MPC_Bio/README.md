Name of Quantlet: Multi-Party Computations for Machine Learning over Genomic Data

Published in: DEDA-HUB-Quantlets

Description: Implementation notebook for privacy-preserving genomic machine learning using
Multi-Party Computation (MPC). Builds a 3-party additive secret-sharing simulator from scratch
and implements allele frequency estimation, GWAS univariate regression, and Polygenic Risk Score
(PRS) in both vanilla NumPy and MPC variants. Uses 1000 Genomes chromosome 22 data (with
synthetic fallback). Benchmarks MPC overhead in runtime, communication bytes, and round
complexity across a sweep of sample sizes and SNP counts.

Keywords: multi-party computation, MPC, genomic data, GWAS, polygenic risk score, secret sharing,
Beaver triples, privacy-preserving machine learning, allele frequency, SNP, 1000 Genomes,
bioinformatics, cryptography, secure computation, communication complexity, performance
benchmarking, visualization

Author: Wolfgang Karl Härdle, Pavel Shibaev
