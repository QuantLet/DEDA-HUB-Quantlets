Name of Quantlet: End to End Bank Fraud Detection Web App

Description: 'An end-to-end machine learning pipeline developed to detect fraudulent financial transactions. The project analyzes a dataset of 6.3 million transactions, addressing high class imbalance (0.13% fraud rate) by employing optimized sampling and threshold tuning (0.91 confidence threshold). It utilizes an XGBoost model optimized for the Precision-Recall trade-off to provide high-quality, actionable fraud alerts for investigative teams, moving beyond simple accuracy metrics.'
